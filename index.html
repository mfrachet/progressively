# Welcome to the Rollout documentation

Rollout is an Open Source feature flags solution (opens new window) built on top of Remix (opens new window), Nestjs (opens new window) and Postgres (opens new window).

On this website, you will find information on how to start the service and how to configure your applications to use it.
